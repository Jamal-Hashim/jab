# 💼 JAB Soft – Software Company Website

JAB Soft is a modern and responsive **software company website** built using **HTML, CSS, and JavaScript**.  
This project showcases a professional business website design that highlights company services, projects, and contact information.

The website is designed with a clean layout, smooth animations, and a user-friendly interface to represent a real-world IT company.

---

## 🌟 Features

- 🏠 **Home Page (Index)** with company introduction  
- 📖 **About Page** describing the company and vision  
- 🛠️ **Services Page** showcasing offered services  
- 💻 **Projects Page** to display completed work  
- 📞 **Contact Page** with company details and inquiry section  
- 🎨 Modern and professional UI design  
- ⚡ Smooth animations and hover effects  
- 📱 Fully responsive design (Mobile + Desktop)  
- 🧭 Easy navigation menu  

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactivity and functionality  
- **Font Awesome** – Icons  
- **Responsive Web Design**
